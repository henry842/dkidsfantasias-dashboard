# DKidsFantasias Dashboard Executivo de Vendas

Dashboard interativo desenvolvido em **Streamlit** para análise de vendas, produtos, pagamentos e previsão de faturamento.

## Funcionalidades

- 📊 **Produtos:** Ranking de vendas, ticket médio e categorias
- 💳 **Pagamentos:** Formas de pagamento, ticket médio por método
- 🗓️ **Temporalidade:** Evolução diária, semanal e mensal
- 📈 **Previsão de Faturamento:** Modelo XGBoost com intervalos de confiança e classificação de precisão

## Tecnologias

- Python 3.10+
- Streamlit
- Pandas
- Altair
- Scikit-learn (XGBoost)
